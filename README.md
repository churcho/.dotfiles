# .dotfiles

## Install

Clone this repository into your home directory.

```shell
$ cd ~
$ git clone git@github.com:mhanberg/.dotfiles.git
```

To bootstrap a new machine, run the `install` script.

```shell
$ ~/.dotfiles/install
```

To only install the dotfiles, run the `rcup` script.

```shell
$ ~/.dotfiles/rcup
```
